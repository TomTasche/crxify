crxify

Quickly pack your Chrome Extensions

View project onGitHub

Say you're on a Chromebook, developing the next blockbuster Chrome Extension. There's no way to write extensions locally, so you're probably using c9.io. But how do you test it? Using crxify, of course!

Shut up and pack!

Restrictions / Requirements:

  • You have to provide an URL pointing to a .zip
  • The root of the archive has to be a directory, which contains your manifest.json
  • for example: announcify.zip/somefolder/manifest.json
  • Give me a few seconds (or minutes, depending on the size of your repository)
